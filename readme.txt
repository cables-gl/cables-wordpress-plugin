Integration of the polyshapes.io into Wordpress

== Description ==

Plugin provides an interface to manage polyshapes.io backgrounds wordpress and
integrate them into yout theme

this is a very rough first version, needs changes.

== Installation ==

1. Upload `polyshapes-wpplugin/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Login as an Adminuser
4. Click on "Polyshapes"
5. fiddle around
